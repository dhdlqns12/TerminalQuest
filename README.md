## 1. 프로젝트 소개

TerminalQuest(터미널퀘스트)는 콘솔 터미널 화면에서 사용자 입력을 통해 다양한 RPG를
Text로 즐기는 게임입니다. 모험, 던전, 강화 등 여러가지 RPG 요소를 즐겨보세요!
